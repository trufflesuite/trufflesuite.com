# HelloWorldBox

## Download Box
```sh
truffle unbox yehia67/helloworldbox
```

## Installation

Install dependencies 
```sh
npm install
```
OS X & Linux:

```sh
truffle develop
compile
migrate
```
Open Another Terminal
```sh
npm run dev
```
To deploy on ropsten
```sh
deploy --network ropsten-infura
```
